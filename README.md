# Sales-Performance-Dashboard-Power-BI-

📊 Sales Performance Dashboard (Power BI)

##  Overview
This project presents an interactive Power BI dashboard designed to analyze sales data and generate actionable business insights. It focuses on revenue trends, regional performance, and product-level analysis.

##  Objectives
- Analyze overall sales performance
- Identify top-performing products and regions
- Track revenue trends over time
- Enable data-driven decision making

##  Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel / CSV dataset

##  Key Features
- Interactive dashboard with slicers and filters
- KPI metrics (Total Sales, Profit, Growth)
- Region-wise and product-wise analysis
- Time-series sales trend visualization

##  Insights
- Identified high revenue-generating regions
- Highlighted top-performing products
- Detected seasonal trends in sales data
